Intro to CLI Workshop

Usually, we can only click or drag so many things on our computer at a particular time. Using text commands, we can get a lot of work done faster. The command-line is a powerful, text-based interface to tell computers what you want to do.

By the end of this workshop, you will be able to see your files, modify or move your files, install and run programs all on the command line. More importantly, you will never be intimidated by that black window with the blinky prompt ever again!

“Never be intimidated by that black window with the blinky prompt ever again!”

Intro to
Git/Github

GitHub is a code hosting platform for collaboration. It lets you work on projects with people anywhere in the world. Part of the beauty is, you don’t need to know how to code to use GitHub. You can use it to help with coordinating and organizing your projects.

By the end of this workshop, you will be able to create a project, manage your project, and collaborate on someone else’s project on GitHub.

“Work on projects with people anywhere in the world with tools for coordinating and organizing your projects”

Intro to
R Workshop

You want to solve problems with data. You’ll need to be ready to go to town in 3 core skill areas: data manipulation, data visualization, and statistics. You’ll need to learn a programming language, ideally just _one_ language. That’s where R comes in.

What’s R? R is an open source programming language and software environment for statistical computing and graphics. Let's get you doing cool things with R and RStudio.

“You want to solve problems with data. Meet R, your new secret weapon.”

Intro to Web Scraping

Many interesting projects start with the same raw material, data. Web scraping is a great way to grab the data you want in a useable form.

By the end of this workshop, you will know how to search for the data you want, make a simple web scraper, and save the data to a spreadsheet on your computer.

“Search and grab data from the internet for your personal and civic hacking projects”
